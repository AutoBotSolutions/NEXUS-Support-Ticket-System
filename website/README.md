# NEXUS Support System Website

This is the GitHub Pages website for the NEXUS Support System project.

## Local Development

To run the website locally:

1. Clone the repository
2. Navigate to the website directory
3. Open `index.html` in a web browser

Or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx http-server

# Using PHP
php -S localhost:8000
```

## Deployment

This website is configured for GitHub Pages. To deploy:

1. Push changes to the `gh-pages` branch or main branch
2. Enable GitHub Pages in repository settings
3. Select source as `gh-pages` branch or main branch with `/docs` folder

## Theme

The website features a sci-fi/cyberpunk theme with:
- Neon cyan and magenta color scheme
- Animated grid overlay
- Particle effects
- Glowing text effects
- Smooth animations and transitions

## Files

- `index.html` - Main landing page
- `styles.css` - Sci-fi themed CSS
- `script.js` - Interactive JavaScript
- `_config.yml` - GitHub Pages configuration

## Customization

To customize the website:

1. Edit `index.html` for content
2. Modify `styles.css` for styling
3. Update `script.js` for interactivity
4. Change `_config.yml` for GitHub Pages settings
